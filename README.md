## 1. 프로젝트 목표

- 아동의 미디어 이용 데이터를 기반으로 **이용 패턴을 분석하고 유형화하는 데이터 분석 시스템을 구축했다.**
- 단순한 통계 분석을 넘어 **군집 분석과 Explainable AI(XAI)** 를 활용해 이용 유형을 해석하고 설명할 수 있도록 설계했다.
- 분석 결과를 사용자에게 전달하기 위해 **웹 기반 인터페이스를 구현했다.**

<br>

---

## 2. 구현 및 기술 스택

### Frontend

- **Streamlit**을 활용하여 웹 기반 인터페이스를 구현했다.
- 사용자가 미디어 이용 정보를 입력하면 분석 결과를 확인할 수 있도록 **인터랙티브한 대시보드 형태로 구성했다.**
- 분석 결과를 시각적으로 보여주기 위해 다양한 **데이터 시각화 그래프**를 활용했다.

<br>



### Explainable AI (XAI)

군집 분석 결과를 해석하기 위해 **Explainable AI 기법을 적용했다.**

**Surrogate Model**

- Random Forest 모델을 활용해 군집 결과를 설명하는 대체 모델을 구축했다.

**Permutation Importance**

- 군집 분류에 영향을 미치는 주요 변수의 중요도를 분석했다.

**SHAP Value**

- 개별 데이터가 특정 군집에 속하게 된 이유를 설명하기 위해 SHAP 기반 해석을 수행했다.

이를 통해 단순한 군집 결과가 아니라 **각 이용 유형이 형성된 원인을 설명할 수 있도록 했다.**

<br>

---

## 3. 결과

- 아동의 미디어 이용 행동을 기반으로 **여러 이용 유형을 데이터 기반으로 도출했다.**
- XAI 분석을 통해 **각 유형을 결정하는 주요 행동 패턴을 해석할 수 있었다.**
- Streamlit 기반 웹 인터페이스를 통해 **사용자가 직접 데이터를 입력하고 분석 결과를 확인할 수 있는 프로토타입 시스템을 구현했다.**
